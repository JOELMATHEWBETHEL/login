This is a simple login done using PHP,BootStrp,jQuery and css.Database used is added as an sql file in the repository.Entire functionality of the login,signup and logout are written as modules in function.php. The login page design code is in be login.php and the signup page design at signup.php. On successful login, the login will be redirected to welcome.php. The name of database is 'log' and it contains 2 tables namely 'signup' and 'login' which is used to hold the details of users signup and sign in. The sql of the tables used in the database is available in 'log.sql'
